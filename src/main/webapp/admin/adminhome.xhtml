Welcome to Hostel Management System Admin


Hostel Ballot - Ballot Admin
  • Search for ballot application by ballot status, start date and end date
  • Click on 'View' button to see more details of a student's hostel ballot application

Hostel Ballot - Eligible Ballot Admin
  • Upload list of students who are eligible to ballot for accommodation.
  • Search list of uploaded students who are eligible for ballot

Hostel Application - Hostel Application Admin
  • Search for hostel application by start date, end date, ballot and payment status. The search parameter can be the student's first name, last name or registration number.
  • Click on 'Print' to print a student's hostel application.

Hostel Application - Eligible Student Admin
  • Add single student eligible for hostel application
  • Search for uploaded eligible student by first name, last name or registration number.
  • Click on 'Update' button in the search result to view more details about a student
  • After clicking on Update button, the current page has tabs for updating student registration number, changing student's hostel allocation, changing student type and delete a student's record.

Hostel Application - Hostel Admin
  • View list of hostels in the database
  • View hostels rooms and students who have been allocated each room

Hostel Application - Eligible Student Upload
  • Upload list of students who are eligible to apply for hostel accommodation.
  • Search list of uploaded students who are eligible for hostel accommodation

Pin Admin - Check Pin Status
  • Enter pin serial number to see the current status of the pin

Pin Admin - Pin Upload
  • Upload pin that will be used by students to pay for hostel application in the portal.
